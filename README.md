# Football Tactical Analysis using Simulated Data

This repository contains 10 football tactical analysis prototypes developed using
simulated positional and event data aligned with professional tracking datasets.

The project focuses on:
- Team structure and shape
- Pressing intensity
- Space occupation
- Transition play
- Tactical xG interpretation
- AI-based match narratives
- Player role effectiveness
- Opponent weakness detection
- Compactness analysis
- Set-piece tactical impact
